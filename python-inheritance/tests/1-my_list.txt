======================
How to Use 1-mylist.py
======================

This module defines a list of integers using the `MyList` class.

===========================
# main file: 
===========================
#!/usr/bin/python3
MyList = __import__('1-my_list').MyList

my_list = MyList()
my_list.append(1)
my_list.append(4)
my_list.append(2)
my_list.append(3)
my_list.append(5)
print(my_list)
my_list.print_sorted()
print(my_list)

===========================
